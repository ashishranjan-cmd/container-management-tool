About App

Container Management Tool

Based on DevOps(Development & Operations) technology. There are various tools in DevOps field but one of the most useful tool is Docker. Meanwhile if you are not familiar with DevOps technology, you might be guessing what the hell is docker?

So, what is docker?

Docker is a set of platform as a service products that use OS-level virtualization to deliver software in packages called containers.

Now you know briefly about the docker. Lets see, How this app works?

Function Buttons

List Containers :
~ List all the containers that is running as well as stopped.
List Images :
~ List all the images available that can be used to launch a container.
Start Container :
~ Start a stopped container.
Stop Container :
~ Stop a running container.
Remove Containers :
~ Remove a container when the work on the container is over.
Execute Container :
~ Run a specific command in a container.
Reset :
~ Reset all the previous work on the app.
